## 一、项目背景与研究目标

### 1.1 研究背景

近年来，全球经济面临诸多不确定性因素：
- **关税政策**：中美贸易战、钢铝关税、电动车关税等
- **贸易战**：美中、美欧贸易摩擦
- **区域冲突**：俄乌战争、巴以冲突等

这些政策事件对全球金融市场产生重大影响，且影响具有以下特点：
1. **即时效应**：部分政策立即影响市场
2. **延迟效应**：部分政策需要时间传导
3. **跨市场传导**：不同市场、不同时间区存在领先-滞后关系

### 1.2 研究目标

本研究旨在扩展现有的MuSA (Multimodal Sentiment-based Adaptive) 框架，构建一个能够：

1. **感知政策事件**：自动识别关税、贸易战、区域冲突事件
2. **多LLM情感分析**：对比FinBERT、GPT、DeepSeek等模型在金融情感分析上的效果
3. **跨时区分析**：分析不同时区市场的响应差异和套利机会
4. **智能投资策略**：基于以上分析生成投资组合

### 2.2 现有算法

| 算法 | 说明 | 状态 |
|------|------|------|
| DC | 基础方向变化算法 | ✅ 已实现 |
| DC-SentimentMulti | 多源情感+DC | ✅ 已实现 |
| MASA-dc | DC + RL控制器 | ✅ 已实现 |
| MASA-mlp | MLP市场观察器 + RL | ✅ 已实现 |
| MASA-lstm | LSTM市场观察器 + RL | ✅ 已实现 |
| TD3-PR | 纯RL (Profit优先) | ✅ 已实现 |

### 2.3 核心组件

**市场观察器 (Market Observer)**：
- 输入：股票价格数据、市场指数
- 输出：隐藏向量 (hidden_vector)、风险参数 (lambda, sigma)
- 模型：MLP_1, LSTM_1, DC_1, SentimentMulti_1

**RL智能体 (TD3)**：
- 策略网络：基于隐藏向量生成股票权重
- 价值网络：评估当前组合的预期收益

**控制器 (Controller)**：
- 方法：CBF (Control Barrier Function) 优化
- 目标：在约束条件下最大化收益

## 五、跨时区分析方案

### 5.1 核心思路

**时区套利的本质**：利用全球市场开盘时间差异，在先开盘市场获得信息后，在后开盘市场进行交易。

```
全球股市开盘顺序（UTC时间）：

东京 → 上海/香港 → 伦敦 → 纽约
 │       │          │       │
 ▼       ▼          ▼       ▼
00:00   01:30      08:00   14:30
```

### 5.2 具体策略

#### 策略1：事件驱动型套利

```
Step 1: 美国政策公布（如关税政策）
Step 2: 观察美股反应方向
Step 3: 在亚洲期货/ETF市场提前布局
Step 4: 亚洲开盘后平仓获利
```

#### 策略2：隔夜持仓套利

```
Day 1 (美国时间):
├── 14:00  政策事件发生
├── 16:00  美股收盘，确定方向
└── 17:00  买入亚洲ETF期货

Day 2 (亚洲时间):
├── 09:30  亚洲开盘，跟随美股方向
└── 11:00  平仓获利
```

### 5.3 可用交易工具

| 类型 | 产品 | 说明 |
|------|------|------|
| 亚洲ETF | FXI(中国), EWJ(日本), EWH(香港) | 美股时段可交易 |
| 欧洲ETF | EWG(德国), EWU(英国) | 美股时段可交易 |
| 期货 | 恒指期货, 日经期货 | 夜间交易 |

## 十、常见问题

### Q1: 如何获取金融新闻数据？
A: 可以使用NewsAPI、Alpha Vantage等免费API，或从FinGPT等数据集获取。

### Q2: LLM API调用需要费用吗？
A: 是的，GPT和DeepSeek的API需要付费。建议先使用免费或低成本方案验证。

### Q3: 跨时区策略如何进行回测？
A: 需要收集多市场日线数据，可以使用Backtrader或自行实现回测框架。

### Q4: 消融实验需要多少时间？
A: 取决于实验规模，一般每个配置需要数小时到一天。
